JSON에 대하여
===

Java Script Object Notation의 약자이다.   

JSON 언어를 쉽게 정의를 내리자면, 간결하고 쉽게 데이터를 정의 내리는 방법 중의 하나.   

JSON 언어의 종류는 2가지.

(1) Json 객체

Key: Value로 이루어진 데이터들의 집합

중괄호로 시작해서 중괄호로 끝나고 Key - Value가 쌍을 이루어 들어가게 되는 것이다.   

예시 : 

```swift 
{

"name": "Hong WonGi"

}

```

Key - Value 값을 더 추가해주고 싶으면, ,(comma)를 이용하면 된다.   

```swift 

{

"name": "Hong WonGi"
"age": "28"

}

```
